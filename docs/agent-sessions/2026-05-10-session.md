## Files Changed

### Engine
- `src/engine/statistics.js` — Added 7 new exported functions:
  - `detectWarmupWelch(replications, metricPath, options)` — Welch's graphical method with midpoint-crossing knee detection, moving-average smoothing, ensemble averaging across replications
  - `batchMeansCI(values, batchSize)` — batch-means CI correcting for autocorrelated output; auto-suggests batch size via `suggestBatchSize()`
  - `suggestBatchSize(values, options)` — heuristic: start at sqrt(n), grow until lag-1 autocorrelation of batch means < 0.1
  - `bonferroniCI(comparisons, alpha)` — Bonferroni correction for multiple pairwise comparisons
  - `compareScenarios(scenarioA, scenarioB, metricPaths, options)` — wraps `pairedTConfidenceInterval()` with Bonferroni correction; returns significance flags at 95% and 99%
  - `computeSummaryStats(values)` — skewness, kurtosis, approximate normality check
  - `computePercentiles(values, percentiles)` — linear-interpolation percentile function (R-7 / NumPy method)
  - Plus private helpers: `linearInterpolate()`, `movingAverage()`, `findKnee()`, `lag1Autocorrelation()`, `tCriticalBonferroni()`
